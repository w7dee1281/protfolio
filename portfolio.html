Chat Log Wizard — GTA-Style Chat Log Generator

A web-based tool I built to generate GTA screen-style chat logs with per-line coloring, font control, and export options. It automatically styles messages (radio, whisper, phone, transactions, roleplay/actions) and also lets you manually override colors on selected lines.

Repository Live Demo
  • Per-line color system with automatic rules + manual override (manual always wins).
  • Automatic message styling for Radio, Whisper, Phone, Give/Money, and Roleplay/Action lines.
  • Manual color override by selecting one or multiple lines using checkboxes.
  • Font control (Tahoma, Roboto, Cairo, Tajawal, Poppins, Montserrat).
  • Export as PNG with background or transparent background while preserving exact styling.
  • Hex color codes supported by adding a color code at the start of a line.
